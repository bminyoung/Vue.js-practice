# Use Webpack

## 1. Install node
- LTS(Long Term Support) 설치


## 2. Initialize
- move to directory
`cd 3.숫자야구`

- initialize
`npm init`

※ package.json (버전 관리 파일) 이 생성된다.

## 3. Use Vue
- install vue
`npm install vue`
or
`npm i vue`

- install webpack
`npm i webpack webpack-cli -D`
 (-D: 개발할 때만 사용하겠다. = `--save-dev`)
※ webpack.config.js 이 생성된다.
